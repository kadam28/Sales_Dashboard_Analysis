# Sales_Dashboard_Analysis
This dashboard shows the overall sales performance in a simple and clear way. It helps to understand total sales, profit, and number of orders at one place. You can easily see which products, regions, or categories are doing well. Charts and graphs make it easy to compare data and find trends quickly. 
📊 Sales Dashboard 2025 – Project Description

This project is a Sales Dashboard created to analyze and visualize sales data in a simple and clear way. It helps to understand important business information like total sales, profit, and number of orders.

The dashboard shows data using charts and graphs, which makes it easy to find trends and patterns. It also helps to compare performance across different categories, products, and regions.

With this dashboard, users can quickly identify which products are performing well and which areas need improvement. It supports better decision-making by providing clear insights from data.

🛠️ Tools & Skills Used
Tableau (for dashboard creation)
Data Visualization
Data Analysis
Business Insights
🎯 Purpose of the Project

The main goal of this project is to practice data analysis and visualization skills and to present sales data in an easy-to-understand format.
